# Workout Tracking Mobile App

This project is a mobile workout tracking application built to explore
frontend development concepts using React and Expo.

The goal of the project was to better understand component-based UI design,
state management, and mobile app workflows by building and modifying a
functional application.

---

## Features
- Log workouts, exercises, sets, and reps
- Support for multiple workout splits
- Calendar-based view of completed workouts
- Simple, mobile-friendly user interface

---

## Tech Stack
- React
- Expo
- JavaScript
- Git

---

## Purpose
This project was developed as a **learning exercise** to:
- Understand React component structure and state flow
- Explore Expo-based mobile development
- Practice modifying and extending an existing codebase
- Gain familiarity with frontend application architecture

---

## Getting Started

### Install dependencies
```bash
npm install
```

### Run the app
```bash
npx expo start
```

Then press `i` for the iOS simulator, `a` for Android, or `w` for the web preview.
